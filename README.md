# Dashboard - тестовое задание на мидла.

## Список технологий

- React
- TypeScript
- Styled Components
- React router
- Api by React query
- Formik + Yup
- Отрисовка графиков Recharts

## Запуск

Приложение построено на create-react-app

- скачать
- установить зависимости <code>npm i</code>
- запуск приложения <code>npm start</code>
